# KhetSeGhar

This repository contains the source code for the landing page of KhetSeGhar, a digital platform designed to connect villages directly to housing societies for bulk grocery procurement. The project aims to create a transparent, efficient, and affordable farm-to-society supply chain.

The live project can be viewed at [khetseghar.in](http://khetseghar.in).

## 🌾 About the Project

KhetSeGhar is building a transparent farm-to-society supply chain where housing societies can procure grains, pulses, and other agricultural products directly from verified Village Hubs. The platform eliminates unnecessary intermediaries, ensuring fair prices for farmers and affordable, high-quality produce for consumers in urban housing societies.

## 🚀 Our Vision

The core vision of KhetSeGhar is to reinvent bulk grocery sourcing with technology and fairness.

*   **Fair Pricing**: Helping farmers receive better value by removing unnecessary intermediaries.
*   **Direct Procurement**: Connecting residential housing societies directly with producing villages.
*   **Bulk Supply**: Tailored specifically for apartment associations and society management.
*   **Transparency**: Providing a platform with verified agricultural inventory and trusted direct sourcing pipelines.

## 👨‍🌾 Platform Modules

KhetSeGhar is designed as an integrated ecosystem with dedicated modules for each stakeholder in the supply chain:

*   **Farmer Hub**: Allows farmers to manage their crops, track yields, and signal harvest readiness.
*   **Village Hub**: Acts as an aggregation center for handling quality checks, inventory management, and local logistics.
*   **Society Hub**: A portal for housing committees to place orders, track bulk shipments, and manage payments.
*   **Admin Control**: A centralized dashboard for platform administrators to handle verification, monitoring, and end-to-end management of the supply chain.
